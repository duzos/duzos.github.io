Death Sound

Pick the sound that plays when you die.

by Duzo

What is it?

A small Forge mod that plays a custom sound when a player dies. Set it with a command and let the deaths roll in.

Usage

Set the death sound with the in-game command:

/setdeathsound <sound>

The chosen sound then plays whenever a tracked player dies.

Requirements

  • Minecraft 1.21.1
  • Forge
