Vortex Manipulator

Save three waypoints. Use them six times.

by Duzo

What is it?

The Vortex Manipulator is a single item that bookmarks locations and jumps you back to them: a personal teleportation device, strictly rationed.

How it works

  • Save waypoints at your current position, up to 3, and you can name them.
  • Shift + right-click to add a new waypoint.
  • Right-click to cycle through your saved waypoints.
  • Press the keybind (B by default) to teleport to the selected waypoint.
  • There's a cooldown between uses, and a hard limit of 6 jumps total.

Requirements

  • Minecraft 1.20.1 (Fabric)
  • Fabric API
  • Java 17+
