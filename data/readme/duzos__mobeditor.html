Mob Editor

Tweak any mob's stats, drops, and effects — straight from a spawn egg.

a commission, by Duzo

What is it?

Mob Editor (listed as Mob Customiser on Modrinth) lets you customise the mobs in your world. Spawn a mob with a custom spawn egg, then edit its stats and behaviour to make it exactly as tough (or as generous) as you want.

What can you edit?

  • Health
  • Speed
  • Attack damage
  • XP dropped
  • Item drops
  • Status effects

Everything is set through the custom spawn egg, so there are no commands to memorise.

Saving

  • Every edited mob is saved and reloaded automatically.
  • Data is stored server-side.
  • Saved on a per-world basis, so each world keeps its own custom mobs.

Requirements

  • Minecraft 1.20.1 (Fabric)
  • Fabric API
  • Java 17+
